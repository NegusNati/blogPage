## The LALIBELA Blog

this project is a practice project with Laravel-9, by understanding the workflow and power of a Laravel back-end.

This project took time but come out very good, it has features for

        - Registration $ Login
        - A simple Dashbord with a table to obeserve the Post info
        - A form to add new Posts with thumbline storage
        - A func to Edit & Delete a post 
        - Well construrcted relationship to implement cascade of users and post
        - A Global search 
        - Catagory organization 
        - A Pagination of Posts
        - Fully Configures Newsletter service with MailChimp
        - and many more

Newsletter Service from 
- [Mailchimp](https://mailchimp.com/developer).
